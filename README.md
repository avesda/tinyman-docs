# Overview

[Tinyman](https://tinyman.org) is a next-generation decentralized AMM on the Algorand blockchain.

{% hint style="info" %}
Tinyman is live on Algorand Mainnet. See [app.tinyman.org](https://app.tinyman.org)
{% endhint %}
